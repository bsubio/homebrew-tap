# Homebrew Tap for bsubio

This is the official Homebrew tap for bsubio CLI.

## Installation

```bash
brew install bsubio/tap/bsubio
```

## About

This tap is automatically maintained by GoReleaser. Formulas are updated with each release of the bsubio CLI.

For more information, visit:
- [bsubio CLI Repository](https://github.com/bsubio/cli)
- [bsubio Website](https://bsub.io)
